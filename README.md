# ABC Company Employee Analysis

## Project Overview
This project analyzes the employee dataset of ABC Company using Python. The dataset contains 458 employee records and was analyzed to understand team distribution, employee positions, age groups, salary expenditure, and the relationship between age and salary.

## Preprocessing
- Loaded the Excel dataset using Pandas.
- Replaced the Height column with random values between 150 cm and 180 cm.

## Analysis Performed
- Employee distribution across teams.
- Employee distribution by position.
- Predominant age group.
- Team with the highest salary expenditure.
- Position with the highest salary expenditure.
- Correlation between age and salary.

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Conclusion
The analysis provides useful insights into employee distribution, salary expenditure, and workforce demographics of ABC Company.
